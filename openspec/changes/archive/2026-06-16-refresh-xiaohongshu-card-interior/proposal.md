## Why

The current Xiaohongshu card interior communicates "notes and tags," but its middle showcase frame feels more generic than platform-specific. A more recognizable Xiaohongshu-focused composition will strengthen the card's identity while keeping it visually aligned with the homepage's dark, polished presentation.

## What Changes

- Refresh the Xiaohongshu card's middle showcase frame to reference a dark editorial cover composition with a circular image-like focal area and a bold Xiaohongshu wordmark treatment.
- Keep the existing outer card shell, carousel footprint, and surrounding text structure unchanged so the update stays contained inside the platform interior.
- Preserve responsive fit by adapting the new composition to the existing interior frame dimensions on desktop and mobile breakpoints.
- Retain visual harmony with the homepage by using restrained red highlights, layered dark surfaces, and simplified supporting detail instead of dropping in a literal source image.
